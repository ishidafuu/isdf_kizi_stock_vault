# dotenvx で .env ファイルを安全に共有する方法 - izanami

## 概要

dotenvx を使えば .env ファイルを暗号化して Git にコミットできる。共同開発者には .env.keys の秘密鍵だけを渡せば、各自が復号して環境変数を取得可能。これで .env 共有問題が解決する

**補足:** Git管理できない機密情報を含む.envファイルを安全に共有する課題を解決し、開発チームでの環境変数運用を効率化する。
